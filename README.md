Perbedaan navigator.push dan navigator.pushreplacement adalah navagitor.push menambahkan halaman baru ke atas stack tanpa menghapus halaman sebelumnya, cocok digunakan pada situasi dimana ketika ada kemungkinan untuk kembali ke halaman sebelumnya. navigator.pushreplacement adalah push halaman ke stack dan menghapus halamana sebelumnya dari stack, cocok digunakan untuk halaman yang jika sudah tidak akan kembali lagi seperti halaman login dll.
2. Scaffold digunakan untuk struktur setiap halaman, appbar digunakan untuk menampilkan judul dan navigasi halaman di bagian atas halaman, dan drawer digunakan untuk menu navigasi di samping halaman.
3. Padding digunakan untuk memberi jarak antar content,SingleChildScrollView memungkinkan tampilan tetap dapat di-scroll ketika konten melebihi tinggi layar,ListView digunakan saat elemen form banyak dan perlu ditampilkan secara vertikal dengan scroll otomatis.Contohnya pada halaman tambah produk, seluruh field input dibungkus dalam SingleChildScrollView, setiap field diberi Padding, dan keseluruhan struktur berada dalam ListView untuk fleksibilitas.
